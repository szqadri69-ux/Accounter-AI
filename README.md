🚀 Accounter-AI: Autonomous Accounting & Compliance SaaS
Accounter-AI ek next-generation financial system hai jo accounting mein manual data entry ko khatam karne ke liye banaya gaya hai. Ye Google Gemini Vision AI ka use karke bills, bank statements aur UPI transactions ko read karta hai aur real-time accounting classification aur audit alerts deta hai.
🌟 Key Features
📸 Live Bill Scanner: Mobile ya laptop camera se physical invoices scan karein. AI turant Date, Items, GSTIN aur Amount nikal leta hai.
📊 Zero-Manual Entry: Bank statements (CSV/Excel) upload karein. Hamara AI engine transactions ko Assets, Liabilities, Income aur Expenses mein classify karta hai.
⚖️ Compliance Guard: Agar turnover limit cross hoti hai toh Section 44AB (Tax Audit) ke liye automatic alert deta hai.
📁 One-Click Export: Categorized data ko Excel/CSV mein download karein, jo Tally mein import karne ke liye ready hota hai.
🔐 Privacy-First: Users sidebar mein apni Gemini API Key daal sakte hain taaki unka data unke control mein rahe.
🛠️ Technology Stack
Frontend: Streamlit (Python-based Web App)
AI Brain: Google Gemini 1.5 Flash (Vision & Text Processing)
Data Handling: Pandas & Openpyxl
OCR: Gemini Vision API
