# 🚀 AI-CA: Autonomous Accounting & Compliance SaaS

**AI-CA** is a next-generation, AI-driven financial ecosystem designed to eliminate manual data entry in accounting. It uses Google's Gemini Vision AI to read bills, bank statements, and UPI transactions to provide real-time accounting, tax categorization, and audit alerts.

---

## 🌟 Key Features

* **📸 Live Bill Scanner:** Use your mobile or laptop camera to scan physical invoices. AI extracts Date, Items, GSTIN, and Amount instantly.
* **📊 Zero-Manual Entry:** Upload bank statements (CSV/Excel). Our AI engine classifies transactions into Assets, Liabilities, Income, and Expenses based on Indian Tax Laws.
* **⚖️ Compliance Guard:** Automatic alerts for **Section 44AB (Tax Audit)** when turnover limits are exceeded.
* **📁 One-Click Export:** Download categorized data in Excel/CSV format, ready to be imported into Tally or for Income Tax filing.
* **🔐 Privacy-First:** Users can provide their own Gemini API Key via the sidebar for complete data control.

---

## 🛠️ Technology Stack

- **Frontend:** [Streamlit](https://streamlit.io/) (Python-based Web App)
- **AI Brain:** [Google Gemini 1.5 Flash](https://aistudio.google.com/) (Vision & Text Processing)
- **Data Handling:** Pandas & Openpyxl
- **OCR:** Gemini Vision API
